# CCC Plastics Limited – Project Instructions

## Project Overview
This is a static website for **CCC Plastics Limited**, a modern redesign of the company's web presence.
Built with plain HTML, CSS, and JavaScript — no frameworks or build tools required.

## File Structure
- `index.html` — single-file website (HTML + embedded CSS + JS)

## Conventions
- All styles are embedded in a `<style>` block in `index.html`
- All scripts are embedded in a `<script>` block at the bottom of `index.html`
- Design tokens (colors, spacing, etc.) are defined as CSS custom properties in `:root`

## Brand
- **Company name:** CCC Plastics Limited
- **Primary color:** `#1a5ca8` (corporate blue)
- **Accent color:** `#d9860f` (amber gold, used for Sulphur division)
- **Dark base:** `#0d1f3c` (navy)
- **Font:** Inter (Google Fonts)
